# legacy/

本目录存放**已废弃、已替换或不再适用**的文档，保留供回溯参考。

## 适合存放的内容

- 已废弃的旧版架构或设计方案
- 从 `../proposals/` 采纳后失去时效性的原始提案
- 被替换模块的历史逻辑说明

## 存放规则

- 移入时建议在文档顶部标注废弃原因和替代方案链接
- 不要删除仍有参考价值的文档
- 完全没有保留价值的文档可以直接删除，无需移入

## 文档命名规范

移入时在原文件名前加日期前缀，便于追溯：

```text
YYYY-MM-DD-original-name.md
```

示例：

```text
2026-05-01-old-message-format.md
```
